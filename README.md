# ArgeyaCase
 This project is a basic Contact Form Application made with ASP.NET Core Web App(MVC) and MSSQL for Argeya Jr. .Net Developer position.


# Requriements

Name, Surname, Phone, Email, Department (Accounting – Technical Support – Human Resources) (Select Input), Message
-We expect you to develop a contact form application using  [ASP.NET](http://asp.net/)  Core Web App (MVC) and MSSQL with the mentioned fields.
